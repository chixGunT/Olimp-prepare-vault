# 🏷️ CVE (Common Vulnerabilities and Exposures)

> 💡 Система, предоставляющая уникальные идентификаторы для известных уязвимостей в программном обеспечении и аппаратных средствах.

## 📖 Основное понятие
*   **📂 Категория:** База данных уязвимостей
*   **🏢 Организация:** MITRE Corporation
*   **🔄 Связанные понятия:** [[9. CWE (Common Weakness Enumeration)]], [[13. NVD (National Vulnerability Database)]], [[10. CAPEC (Common Attack Pattern Enumeration and Classification)]]

## 📋 Подробное описание
**CVE** служит основным источником для обмена информацией об уязвимостях и их исправлениях. Каждой уязвимости присваивается уникальный идентификатор формата `CVE-ГГОД-НОМЕР`.

### ✨ Ключевые особенности
*   🎯 Уникальные идентификаторы для уязвимостей
*   📚 Основной источник информации об уязвимостях
*   🔄 Основа для других баз данных (NVD)
*   🌍 Международный стандарт

## 🧪 Примеры
*   📌 **CVE-2021-44228** — уязвимость Log4Shell в Apache Log4j
*   📌 **CVE-2017-0144** — уязвимость EternalBlue в SMB
