# 🏷️ Вредоносное ПО

> 💡 Скрытые цифровые диверсанты, которые проникают в компьютер или телефон под видом полезных программ, игр или документов.

## 📖 Основное понятие
*   **📂 Категория:** Программные атаки
*   **🔄 Связанные понятия:** [[11. Ботнет]], [[#🎯 Типы вредоносного ПО]]

## 📋 Подробное описание
**Вредоносное ПО** (Malware) — программы, созданные для тайного проникновения в системы с целью кражи данных, повреждения системы или получения контроля над устройством.

### 🎯 Типы вредоносного ПО
*   **🦠 Вирусы** — самовоспроизводящиеся программы
*   **🐴 Трояны** — маскируются под легитимное ПО
*   **👁️ Шпионские программы** — тайно собирают информацию
*   **💰 Вымогатели** — блокируют данные за выкуп
*   **🤖 Боты** — для создания ботнетов

### 🎯 Способы распространения
*   📧 Поддельные вложения в письмах
*   🌐 Загрузка с поддельных сайтов
*   💾 Зараженные флеш-накопители
*   📱 Вредоносные приложения

## 🧪 Примеры
*   📌 Игра, которая крадет пароли
*   📌 Документ Word с макровирусом
*   📌 Программа-вымогатель, блокирующая файлы
*   📌 Приложение, включающее камеру без ведома пользователя
